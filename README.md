A simple Java console chatbot created as part of a learning project.  
The bot interacts with the user in several steps:
- Greets the user and asks for their name.
- Guesses their age based on remainders from division by 3, 5, and 7.
- Counts up to a number provided by the user.
- Tests the user's programming knowledge with a mini-quiz.
